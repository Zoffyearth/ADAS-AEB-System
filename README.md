# ADAS AEB 自动紧急制动系统

> 基于 Python + YOLOv8 的自动紧急制动系统（Autonomous Emergency Braking），包含目标检测、距离估计、TTC风险评估、车辆动力学仿真和Web可视化交互界面。

## 项目简介

本项目实现了一个完整的 ADAS 自动紧急制动系统，采用经典的三层架构设计：**感知层 → 决策层 → 控制层**。系统通过摄像头视频输入，实时检测前方车辆、行人和交通信号灯，估算距离与相对速度，计算碰撞时间（TTC），并根据风险等级触发预警或紧急制动。

## 功能特点

### 🎯 目标检测
- 基于 YOLOv8 深度学习模型
- 支持三类目标检测：车辆、行人、交通信号灯
- 实时检测框标注与置信度显示

### 📏 距离估计
- 单目视觉测距模型（D = K / h_bbox）
- 滑动平均滤波平滑处理
- 支持多目标同时测距

### ⚠️ 风险评估
- TTC（Time To Collision）碰撞时间计算
- 三级风险分级：安全 / 预警 / 紧急制动
- 可配置 TTC 阈值参数

### 🚗 车辆动力学仿真
- 一维纵向动力学模型
- 考虑制动力、空气阻力、滚动阻力
- 制动系统一阶响应延迟
- 支持自定义车重、摩擦系数、初速等参数

### 🌐 Web 交互界面
- 视频上传与实时检测画面
- TTC / 距离 / 车速 / 系统状态 仪表盘
- 四条实时曲线图（距离、TTC、车速、制动信号）
- 车辆跟车动画示意
- 底部独立制动仿真面板（5种分析模式）

## 快速开始

### 环境准备

```bash
# 安装依赖
pip install -r requirements.txt
```

### 启动系统

```bash
# 方式一：启动 Web 界面（推荐）
python run.py

# 方式二：命令行模式，处理视频并生成结果
python run.py --cli your_video.mp4 --output data/output

# 方式三：仅生成制动仿真曲线
```

启动后浏览器访问：**http://localhost:8000**

1. 打开 Web 界面
2. 点击「上传视频」选择测试视频（MP4格式）
3. 点击「开始检测」启动实时检测
4. 观察检测画面、TTC仪表盘和实时曲线
5. 在底部「制动仿真」区域调整参数并运行仿真

## 核心算法

### 距离估计
```
D = K / h_bbox
```
- K：标定常数（~2200 m·px）
- h_bbox：检测框高度（像素）

### TTC 碰撞时间
```
TTC = D / v_rel
```
- D：到前车距离（m）
- v_rel：相对速度（m/s），正值表示接近

### AEB 决策规则
| TTC 范围 | 状态 | 动作 |
|---------|------|------|
| TTC ≥ 2.0s | SAFE 安全 | 保持行驶 |
| 1.0s ≤ TTC < 2.0s | WARNING 预警 | 碰撞预警，轻度制动 |
| TTC < 1.0s | BRAKE 紧急制动 | 全力制动 |

### 车辆动力学
- 制动力：F_brake = signal × m × a_max
- 空气阻力：F_drag = 0.5 × ρ × Cd × A × v²
- 滚动阻力：F_roll = Cr × m × g
- 总减速度：a = ΣF / m

## 仿真场景

系统支持多种典型场景测试：

1. **前车正常行驶** — 安全跟车，无制动触发
2. **前车突然减速** — TTC骤降，触发预警 → 紧急制动
3. **行人突然进入车道** — 检测到行人，紧急避撞
4. **红绿灯变化** — 检测到红灯，主动停车制动

## 制动仿真分析模式

1. **单次仿真** — 给定初速和制动时机，输出速度/减速度/距离曲线
2. **多速度对比** — 同一制动时机，不同初速的制动效果对比
3. **环境场景** — 干燥/雨天/积雪/重载/空载 5种路况对比
4. **刹车距离分析** — 全速度段刹车距离曲线
5. **TTC阈值分析** — 不同初始距离下的TTC和避撞效果
